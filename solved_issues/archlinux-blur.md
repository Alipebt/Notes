# Arch Linux 下毛玻璃特效的最佳搭配

## 1. 主题

安装`kvantum`

```shell
yay -S kvantum 
```

在[这里](https://www.pling.com/browse?cat=123&ord=ratin)下载[Fluent kvantum theme](https://www.pling.com/p/1499836)主题(可通过`ocs-url`下载)

在`kvantum`中安装并使用该主题。

全局主题使用`Layan`

## 2. 终端

直接在终端外观配置中改变窗口透明度

## 3. 脚本

毛玻璃脚本（将纯透明转化为毛玻璃特效）：

```shell
设置 --> 窗口管理 --> 桌面特效 --> Batter Blur
```

平铺脚本：

参照[这里](https://github.com/lingtjien/Grid-Tiling-Kwin)的`Grid Tiling`脚本

```shell
设置 --> 窗口管理 --> KWin --> Grid Tiling
```
