## 此文件夹路径应为 ~/
## 其中.config文件夹路径为 ~/.config

### ./
`.zshrc`中包含了对`zim`框架的配置
若出现`warning: completion was already initialized before completion module.` 
可尝试删掉`/etc/zsh/zshrc`文件


`.zshrc` 配置类fzf相关内容 

fzf 快速搜索文件/目录
- 依赖`fd`：可搜索隐藏文件
- 依赖`bat`：高亮输出内容
- 依赖`lsd`: 树状输出目录
